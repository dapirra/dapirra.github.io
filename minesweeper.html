Assembly Minesweeper:

View on Github

Description:

This Minesweeper game was created with the MIPS assembly language and runs on Stony Brook's customized version of the MARS Emulator

How it works:

  • Works similar to minesweeper on Windows
  • The cursor can be moved with the WASD keys
    • R is used to reveal an area
    • F is used to mark a square with a flag
  • Winning the game occurs when all bombs are flagged
