APNG Thing

View on Github

Description:

This is an extremely simple command line program (less than 50 lines of python) that's used to adjust the frame rate of an APNG file. Note that changing the frame rate will in effect, change the playback rate of the animation.

Usage:

apng_thing.py sample.png 25

Sample APNG:

Feel free to test out the script on this sample if you'd like.

May display a still image if apng is not supported in your browser.
