Edge Only Plugin

Download Plugin View Source

About:

This is a plugin that I created for Paint.NET that will only leave the edge of an object when run, and will erase everything else on the inside. In Paint.Net, an object is defined as a group of pixels surrounded by transparent pixels. The thickness of the remaning edge can also be controlled in the user interface.

How to install:

Either run the included bat file or manually copy the dll file into Paint.NET's effects folder located in its installation folder (usually C:\Program Files\paint.net\Effects).
